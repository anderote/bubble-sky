# bubble-sky

A local, moddable Minecraft (Fabric 1.21.6) server that doubles as an AI-agent playground:

- **Write Java mods heavily** in a Fabric dev workspace.
- **Claude Code drives a character** in-world via a Mineflayer-based MCP server.
- **Autonomous LLM bots** (mindcraft) live in the world alongside you.

Status: bootstrapping. Design lives in `docs/superpowers/specs/`.
